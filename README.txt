Web Crawler of Sina News website is writen in Jupyter notebook.
Provide 2 kinds of file to read:
1.web_crawler.ipynb can be opened by Jupyter notebook (suggested).
2.web_crawler.py can be opened by Pycharm.

How to run:
Modify parameters of function getNewsTotal(start, end) to decide the page range you want to collect.

In demo, use getNewsTotal(11,13) to show results of page 11&12.
Demo result is show in news_result.xlsx
